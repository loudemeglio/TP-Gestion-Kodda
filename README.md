# TP-Gestion-Koddaß

### Requisitos
- Docker
- Docker Compose

### Levantar la aplicación

1. Navega a la carpeta `src`:
   ```bash
   cd src
   ```

2. Levantar los contenedores:
   ```bash
   docker-compose up -d --build
   ```

3. Abrir el navegador e ingresar a la siguiente URL:
   ```
   http://localhost:3000
   ```
